# UEFI 管理器

 UEFI 启动项图形化管理工具，运行于飞牛(fnOS) NAS 系统。

## 功能特性

- **启动项查看** — 显示所有 UEFI 启动项及其状态（当前/下次启动项、激活状态）
- **启动顺序管理** — 修改启动项顺序
- **下次启动设置** — 设置或清除下次启动项（一次性生效）
- **启动项增删改** — 新增、编辑、删除 UEFI 启动项
- **激活/禁用** — 单独激活或禁用指定启动项
- **磁盘分区浏览** — 自动识别系统磁盘及分区，辅助创建启动项
- **备份与恢复** — 创建启动配置备份，支持从备份恢复（自动创建安全备份）
- **主题适配** — 自动跟随飞牛系统亮色/暗色主题

## 系统要求

- 飞牛 fnOS 系统（x86_64 架构）
- UEFI 固件模式（非 Legacy BIOS）
- efibootmgr（安装时自动检测并安装）
